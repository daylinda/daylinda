## 🚀 Things I've built

### 🤖 [RPA-ApplicationProcess](https://github.com/daylinda/RPA-ApplicationProcess)
Tired of filling in the same job application form over and over? Same. So I built a pipeline to automate it — Chrome extension + .NET API + Firebase + UiPath, targeting SEEK.

`C#` `.NET` `TypeScript` `Firebase` `UiPath` `Chrome Extension`

### 🛒 [GroceryPrice-Comparer](https://github.com/daylinda/GroceryPrice-Comparer)
Checks prices across grocery stores and tells you where to buy each item for less — with location-aware store suggestions. A very practical tool born from being a broke uni student.

`Price Comparison` `Location-aware` `Grocery`
